# React Workshop

### Prerequisities

- Yarn or NPM

### Installation

```
yarn install
```

or

```
npm install
```

### Test

```
yarn test
```

or

```
npm run test
```

### Development

Start a developement server
```
yarn dev
```

or

```
npm run dev
```

### Production


Build project to the `dist` folder
```
yarn build
```

or

```
npm run build
```

Then you can serve it with your usual production server
